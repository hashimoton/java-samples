***************
Java Package
***************

A Java class belongs to a package.

==============
Structure
==============

::

  ├── README.rst
  └── src
      └── local
          └── demo
              └── Packaged.java


=============
Exercise
=============

Compile
------------

::

  $ cd src
  $ javac local/demo/Packaged.java

Run
-------------

::

  $ cd src
  $ java local.demo.Packaged
  Hello!
  $
